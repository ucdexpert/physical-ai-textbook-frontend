# Additional Resources and References

## Overview
This section provides a curated list of external resources, official documentation, and further reading to deepen your understanding of Physical AI & Humanoid Robotics. These references complement the course material, offering opportunities for continued learning, troubleshooting, and staying updated with the latest advancements in the field.

## Categories:

### Official Documentation
- **ROS 2 Documentation**: The official and most comprehensive resource for all things ROS 2. Essential for understanding core concepts, APIs, and tools.
    - [docs.ros.org](https://docs.ros.org/)
- **Docusaurus Documentation**: Official documentation for the Docusaurus framework. Useful for advanced configuration, theme customization, and plugin development.
    - [docusaurus.io/docs](https://docusaurus.io/docs/)
- **NVIDIA Isaac Sim Documentation**: Official guides, tutorials, and API references for NVIDIA Isaac Sim.
    - [docs.omniverse.nvidia.com/isaacsim/latest/index.html](https://docs.omniverse.nvidia.com/isaacsim/latest/index.html)
- **NVIDIA Isaac ROS Documentation**: Official documentation for Isaac ROS packages, including installation, tutorials, and API references for accelerated ROS 2 modules.
    - [docs.nvidia.com/isaac/ros/index.html](https://docs.nvidia.com/isaac/ros/index.html)
- **Nav2 (ROS 2 Navigation Stack) Documentation**: Comprehensive documentation for configuring and using the Nav2 stack.
    - [navigation.ros.org](https://navigation.ros.org/)
- **OpenAI Whisper Documentation**: Official API documentation and model details for OpenAI Whisper.
    - [openai.com/research/whisper](https://openai.com/research/whisper)
- **OpenAI API Documentation**: General API documentation for various OpenAI models, including GPT.
    - [platform.openai.com/docs/api-reference](https://platform.openai.com/docs/api-reference)
- **Intel RealSense SDK (librealsense)**: Documentation and SDK for Intel RealSense cameras.
    - [github.com/IntelRealSense/librealsense](https://github.com/IntelRealSense/librealsense)

### Books and Research Papers
- **"Probabilistic Robotics" by S. Thrun, W. Burgard, D. Fox**: A foundational textbook for mobile robotics, covering Bayes filters, SLAM, localization, and mapping.
- **"Robot Dynamics and Control" by M. Spong, S. Hutchinson, M. Vidyasagar**: Classic textbook for understanding robot kinematics, dynamics, and control.
- **"Deep Learning" by I. Goodfellow, Y. Bengio, A. Courville**: The go-to textbook for deep learning fundamentals.
- **"Designing Embodied Agents" by H.H. Clark**: Explores the psychological and cognitive aspects of embodied interaction.
- **Recent Robotics/AI Conference Proceedings**: Stay updated by browsing papers from conferences like ICRA, IROS, RSS, NeurIPS, ICML.

### Video Tutorials and Courses
- **ROS 2 Tutorials (Articulated Robotics, The Construct)**: YouTube channels with excellent ROS 2 video tutorials.
- **NVIDIA GTC (GPU Technology Conference)**: Keynotes and sessions often feature Isaac Sim and ROS.
- **Docusaurus Video Guides**: Official or community-made video tutorials for Docusaurus setup and usage.

### Community and Forums
- **ROS Discourse**: The official forum for ROS 1 and ROS 2 users to ask questions, share knowledge, and discuss development.
    - [discourse.ros.org](https://discourse.ros.org/)
- **NVIDIA DevTalk Forums (Isaac Sim, Isaac ROS sections)**: Official NVIDIA forums for support and discussions related to Isaac platforms.
    - [forums.developer.nvidia.com/c/agx-autonomous-machines/isaac/37](https://forums.developer.nvidia.com/c/agx-autonomous-machines/isaac/37)
- **Docusaurus Community**: Forums and Discord channels for Docusaurus users.
- **Stack Overflow**: General programming Q&A site; use `ros2`, `docusaurus`, `isaac-sim`, `humanoid-robotics` tags.

### GitHub Repositories
- **ROS 2 Examples**: Official ROS 2 examples for various communication patterns and features.
    - [github.com/ros2/examples](https://github.com/ros2/examples)
- **NVIDIA Isaac Examples**: Reference examples for Isaac Sim and Isaac ROS.
    - [github.com/NVIDIA-AI-IOT/isaac_ros-dev](https://github.com/NVIDIA-AI-IOT/isaac_ros-dev)
- **Your Project's GitHub**: This course's companion GitHub repository (where this book is hosted) will serve as a continuous reference for code examples and project structure.
